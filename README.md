7-Segment Shield
=======

[![7-Segment Shield](http://ep.yimg.com/ca/I/yhst-27389313707334_2252_124671535)  
*7-Segment Shield*](http://www.gravitech.us/7segmentshield.html)

 - I2C 4 digits 7-segment driver 
 - I2C temperature sensor
 - I2C EEPROM
 - PWM RGB LED
 - Blue PWR LED
 - Reset button
 - All pins breakout
